ExParsec.Error

  • Summary
  • Functions
  • Types

Represents a parse error encountered when executing a parser function.

  • message is the error message.
  • kind is the error kind. nil if the error doesn’t fit into the list of standard error kinds.
  • position is the position in the input data where the error occurred. Can be nil if the input doesn’t support position tracking.
